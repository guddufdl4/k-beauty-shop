# 모바일에서 쇼핑몰 테스트하기

바이어에게 같은 Wi-Fi 환경에서 PC 개발 서버를 휴대폰으로 보여줄 때 사용합니다.

## 1. 개발 서버 실행

프로젝트 루트에서:

```bash
npm run dev
```

`package.json`의 dev 스크립트는 `next dev --hostname 0.0.0.0`으로 설정되어 있어, 같은 네트워크의 다른 기기에서 PC에 접속할 수 있습니다.

터미널에 `Local:`과 `Network:` URL이 표시됩니다.

## 2. PC IP 주소 확인 (Windows)

PowerShell 또는 명령 프롬프트에서:

```bash
ipconfig
```

**무선 LAN 어댑터 Wi-Fi** (또는 사용 중인 어댑터)의 **IPv4 주소**를 확인합니다.

예: `192.168.0.12`

## 3. 휴대폰 브라우저에서 접속

PC와 **같은 Wi-Fi**에 연결한 뒤, 휴대폰 브라우저 주소창에 입력:

```
http://[PC-IP]:3000
```

예: `http://192.168.0.12:3000`

홈, 상품 목록, 장바구니, 결제, 주문 확인 페이지를 모바일 화면에서 확인할 수 있습니다.

## 4. 참고

| 접속 방법 | URL 예시 |
|-----------|----------|
| PC 브라우저 (로컬) | `http://localhost:3000` |
| 휴대폰 (같은 Wi-Fi) | `http://192.168.0.12:3000` |

- PC와 휴대폰이 **같은 Wi-Fi**여야 합니다.
- Windows 방화벽에서 Node/Next.js(포트 3000) 허용이 필요할 수 있습니다.
- 인터넷 공개 URL이 필요하면 **Phase 7**에서 Vercel 등으로 배포하세요.
